KeyFu is a keyword engine. It allows you to attach functionality to keywords and navigate the web faster.

Why

KeyFu was created because I was addicted to Firefox smart keywords but wanted them to:

  • Work in all my browsers (Chrome, Firefox, IE, Opera)
  • Be synced between all my computers and browsers
  • Have advanced functionlity which couldn't be created with Firefox's keywords

When

I've had personal versions of KeyFu running since early 2005, but a public version wasn't available until 2009.
